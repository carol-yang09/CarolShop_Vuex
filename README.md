## 簡介

使用 Vue Cli 、 Vuex 及 Vue Router 製作的購物網站，主要功能有

#### 前台：
* 首頁 Banner 、 商品 輪播
* 管理員登入
* 購物車功能
* 商品列表、分類展示
* 單一商品詳細介紹
* 訂單結帳
* 優惠卷套用
* 結帳表單驗證
* 簡易轉盤遊戲

#### 後台：
* 商品管理
* 客戶訂單管理
* 優惠卷管理
